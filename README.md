#  Full Stack Programs

This repository contains a collection of **basic web pages** and simple full-stack projects built to demonstrate foundational web development skills using HTML, CSS, JavaScript, and possibly backend integration.

These pages can be used as practice material or learning examples for beginners in web development.

This project contains several basic web pages such as:

✔ CSS styled layouts  
✔ JavaScript interactions  
✔ Form handling (if included)  
✔ Basic responsive design (where applicable)

Each folder/page illustrates a different concept of web development.
